# Фермерское хозяйство - ИС учета

## Описание
Система для учета животных, кормов, продукции, прививок и покупателей в фермерском хозяйстве.

## Технологии
- **Backend**: Flask (Python)
- **Database**: PostgreSQL  
- **Frontend**: HTML/CSS/JavaScript
